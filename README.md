# Focus Timer

![Screenshot](PREVIEW.gif)

Simple [pomodoro](https://en.wikipedia.org/wiki/Pomodoro_Technique) timer for the mac touch bar  
25 min work - 5 min rest

- **Tap** to start
- **Double tap** to reset
- **Hold** to close
- **Swipe** to change time
